-Atribute-

lang="id" adalah atribut, lang sebagai nama atribut dan id sebagai nilai atribute

Atribute terdiri dari global atribut dan spesifik atribut

Global atribut seperti, acceskey, class, dir, hidden, id, lang, dll


-Paragraf-

peragraf menggunakan elemen (p)

-Heading-

Heading atau header menggunakan tag elemen (h1) sampai(h6)


Contoh

Bandung

Geografis

Kota Bandung dikelilingi oleh pegunungan, sehingga bentuk morfologi wilayahnya bagaikan sebuah mangkok raksasa

Wisata

Kota Bandung telah menjadi tujuan utama dalam menikmati liburan akhir pekan terutama dari masyarakat yang berasal dari Jakarta sekitarnya.

Farm House Lembang

Lokasi

Berada di jalur utama Bandung-Lembang, Farm House menjadi objek wisata yang tidak pernah sepi pengunjung.

Kondisi

Selain karena letaknya strategis, kawasan ini juga menghadirkan nuansa wisata khas Eropa. Semua itu diterapkan dalam bentuk spot swafoto Instagramable.

Observatorium Bosscha

Memiliki beberapa teleskop, antara lain, Refraktor Ganda Zeiss, Schmidt Bimasakti, Refraktor Bamberg, Cassegrain GOTO, dan Teleskop Surya.

-List-

Terdapat tiga tipe list, unordered list, Ordered lists, description list


Unordered List

menampilkan bullet pada tiap item listnya

Contoh :

  • item 1

  • sebuah heading dalam list

  • sebuah paragraf dalam list


Ordered List

menampilkan angka atau huruf pada tiap item di listnya

Contoh :

  1. Langkah pertama
  2. Langkah kedua
  3. Langkah ketiga
  4. Langkah selanjutnya
  1. Langkah pertama
  2. Langkah kedua
  3. Langkah ketiga
  4. Langkah selanjutnya
